# 🐅 Baghchal – Traditional Nepali Strategy Game

> A modern web-based implementation of Nepal's iconic board game **Baghchal (Tiger and Goats)**, built with React, TypeScript, and Vite.

🌐 **Live Demo:** https://baghchalgame.vercel.app
---

## 📖 Overview

Baghchal (Bagh-Chal) is a traditional Nepali strategy board game played between **4 Tigers** and **20 Goats**. The Tigers attempt to capture goats, while the Goats work together to trap all Tigers and prevent them from moving.

This project brings the classic game to the modern web with an intuitive interface, responsive design, smooth gameplay, and intelligent game mechanics while preserving the authentic rules of the original game.

## 🚀 Getting Started

### Prerequisites

* Node.js 20+
* npm

### Clone Repository

```bash
git clone https://github.com/DipeshR23/baghchal_game
cd baghchal_game
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## 🎯 How to Play

### Tigers 🐅

* Start with 4 Tigers placed at the corners.
* Move along connected lines.
* Capture goats by jumping over them.
* Win by capturing enough goats.

### Goats 🐐

* Place goats strategically on the board.
* Work together to block and trap Tigers.
* Win by preventing all Tigers from making legal moves.
